PROMPT 1 · AMH 법안
AMH 데이터 개방 법안
  1. 아래 프롬프트를 복사
  2. 주로 사용하는 AI(ChatGPT/Gemini/Claude)에 붙여넣기
  3. A4 1장짜리 단일 소스 받기
  4. 받은 텍스트 전체 복사 → NotebookLM에 붙여넣기
당신은 보건복지 정책을 분석하는 정책 리서처입니다.
AMH 검사 데이터 개방 법안 제안서를 위한 단일 소스 문서를 만들어줘.
이 문서는 NotebookLM에 업로드해서 5장짜리 슬라이드덱을 만들기 위한 용도야.
짧고 간결하게, A4 1장 분량으로 작성해줘.

[배경]
- 보건복지부는 가임력 검사 지원 사업으로 AMH 검사비를 지원 중
- 그런데 질병관리청 난임 코호트는 AMH 데이터를 수집하지 않음
- 즉, 검사는 국가가 지원하지만 데이터는 흩어져 사라지는 구조

[문서 구성 — 딱 4개 항목만, 각 3~5줄]
1. AMH란? — 한 줄 정의 + 왜 중요한지 한 줄
2. 문제 — 정책 역설 한 문단 (검사 지원 vs 데이터 미수집)
3. 해외 사례 — 1~2개국 짧게 (영국 UK Biobank 등 데이터 개방 사례)
4. 법안 제안 — 핵심 3줄 (수집 의무화 / 비식별 개방 / 연구 활용)

[규칙]
- 전체 A4 1장 이내 (1,000자 내외)
- 표·그래프·이모지 없이 텍스트만
- 출처는 본문 끝에 3~5개만 간단히
- 슬라이드 5장 (표지 / 문제 / 해외사례 / 제안 / 기대효과) 만들기 좋게 정리

구성:
1. 표지 — 법안 제목 + 한 줄 요약

4. 법안 제안 — 수집 의무화 / 비식별 개방 / 연구 활용

규칙:
